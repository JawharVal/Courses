

# Проект: **Курсы (Courses App)**

## Описание проекта

Это Android-приложение, предназначенное для отображения списка курсов с возможностью взаимодействия с ними. Включает в себя такие функции, как:

* Отображение информации о курсах, полученной с API.
* Показывает курсы с их деталями, такими как название, описание, цена, рейтинг и дата публикации.
* Сортировка курсов по дате добавления.
* Возможность добавления курсов в избранное.

## Требования к задаче

Данный проект был разработан в соответствии с требованиями задачи:

* **Поиск**: Реализовать функционал поиска, чтобы пользователи могли искать курсы.
* **Отображение курсов**: Отображение курсов с полями:

  * `id` — уникальный идентификатор курса.
  * `title` — название курса.
  * `text` — описание курса.
  * `price` — цена курса.
  * `rate` — рейтинг курса.
  * `startDate` — дата начала курса.
  * `publishDate` — дата публикации курса.
  * `hasLike` — флаг, добавлен ли курс в избранное.
* **Особенности UI**:

  * Описание курса обрезается, если текст не помещается в два ряда.
  * Реализована сортировка курсов по дате добавления.
  * Поиск и иконка фильтра — нефункциональные элементы.

## Используемые технологии

* **Kotlin**: Основной язык программирования.
* **Room**: Для работы с локальной базой данных для хранения избранных курсов.
* **Retrofit**: Для получения данных с API.
* **Coroutines**: Для выполнения асинхронных операций.
* **MVVM**: Для разделения логики приложения и пользовательского интерфейса.
* **ViewBinding**: Для удобной работы с представлениями.
* **RecyclerView**: Для отображения списка курсов.

## Структура проекта

* **AppDatabase**: База данных Room для хранения избранных курсов.
* **CourseAdapter**: Адаптер для отображения списка курсов в `RecyclerView`.
* **ViewModel**: Управление данными для UI и логика сортировки курсов.
* **API**: Получение данных о курсах с API и парсинг JSON в список курсов.

## Функции

1. **Отображение курсов**: Отображается список курсов, полученных с API. Каждая карточка курса отображает название, описание, цену, рейтинг и дату публикации.
2. **Избранные курсы**: Пользователь может добавить курсы в избранное. Избранные курсы сохраняются в локальную базу данных.
3. **Сортировка**: Курсы могут быть отсортированы по дате публикации.
4. **Поиск**: В приложении присутствует поле для поиска (нефункциональное, согласно заданию).
5. **UI**: Элементы UI соответствуют макетам из Figma.

## API

Данные о курсах получаются с API по следующему адресу:

* **API Endpoint**: [API курсов](https://drive.usercontent.google.com/u/0/uc?id=15arTK7XT2b7Yv4BJsmDctA4Hg-BbS8-q&e=export&download)

Ответ от API включает список курсов с полями `id`, `title`, `text`, `price`, `rate`, `startDate`, `publishDate` и `hasLike`.

## Установка и настройка

1. **Клонирование репозитория**:

   ```bash
   git clone <url-репозитория>
   cd <папка с проектом>
   ```

2. **Установка зависимостей**:
   Откройте проект в **Android Studio**, и зависимости будут автоматически синхронизированы.

3. **Запуск проекта**:
   Подключите устройство или запустите эмулятор, затем нажмите "Run" в Android Studio.

4. **Настройка базы данных**:
   При первом запуске приложения будет создана база данных `courses.db` для хранения избранных курсов.

## Использование

1. **Главный экран**:

   * На главном экране отображаются все курсы.
   * Можно прокручивать список курсов и видеть их данные.
   * Курс можно добавить в избранное, нажав кнопку **Like**.

2. **Поиск**:

   * Поле для поиска в настоящее время не функционально, как указано в заданиях.

3. **Избранные курсы**:

   * Добавленные курсы сохраняются в локальной базе данных.

## Известные проблемы

* Поиск и фильтрация не реализованы в этой версии, как указано в задании.
* Нет логики для управления состоянием пользователя (например, вход в систему или профили пользователей).
